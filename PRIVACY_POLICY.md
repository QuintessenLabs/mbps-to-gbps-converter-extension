# Privacy Policy for Mbps to Gbps Converter

**Last Updated: June 13, 2026**

This privacy policy governs your use of the Chrome Extension **Mbps to Gbps Converter** ("the Extension"). 

### 1. Data Collection and Usage
The Extension does not collect, store, or transmit any personally identifiable information, browsing history, user activity, or personal data. 

All user configurations (such as your enablement status and page-specific conversion statistics) are stored locally on your device using Chrome's secure storage API (`chrome.storage.local`). This data is never sent to our servers, third parties, or any external network.

### 2. Permissions Required
The Extension requests the minimum permissions necessary to function:
* **`storage`**: Used exclusively to save and load your enablement toggle and keep track of your conversion counters locally.
* **Host Permissions (`http://*/*` and `https://*/*`)**: Used to parse and convert speed notations on web pages you visit. No data regarding the websites you visit is recorded or transmitted.

### 3. Remote Code
The Extension runs entirely offline using local packages. It does not load or execute any remote code, tracking scripts, or external libraries.

### 4. Changes to This Policy
We may update this privacy policy from time to time. Any changes will be posted directly on this page.
